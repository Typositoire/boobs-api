# Boobs as a Service
Flashing boobs legally, inspired by [Dicks API](http://dicks-api.herokuapp.com/)

## Flash some boobs!
Start flashing some boobs [here](https://boobs-api.herokuapp.com/)!

## Usage

Request:
```
GET /v1/boobs/1
```

Response:
```
{"boobs":["( . y . )"]}
```

or the SFW version

Request:
```
GET /v1/boobs/1?sfw=1
```

Response:
```
{"boobs":["(omit)(omit)"]}
```
